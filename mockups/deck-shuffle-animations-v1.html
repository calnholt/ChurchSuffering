Implementation Notes

  • Cross Assembly — 4 directional ingress groups (top/bottom/left/right); each card's start position is off-screen on the matching axis. Phase 1 (~1.1s): ease-out to cross slots. Phase 2 (~0.9s): staggered converge to draw-pile anchor with scale-down.
  • Riffle Shuffle — Split deck entity into left/right stacks with slight rotation; alternating Y-drop with 40ms stagger mimics interleave. Final 200ms: compress stack + subtle bounce.
  • Cascade Peel — Sequential peel using quadratic bezier arcs; fan spread at apex then tuck. Good for shorter decks or subtle reshuffle feedback.
  • All variants should block input briefly (like MillCardSystem) and publish completion before draw animations begin.
